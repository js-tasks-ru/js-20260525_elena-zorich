# ProductForm компонент с отправкой/загрузкой данных с сервера

Необходимо добавить функционал "Drag'n'Drop" в компонент ["ProductForm v1"](taskbook:forms-fetch-api/product-form-v1)
из предыдущего модуля "Взаимодействие с сервером"

!["ProductForm v2"](product-form-v2.gif)

За функционал "Drag'n'Drop" должен отвечать компонент ["SortableList"](taskbook:tests-for-frontend-apps/sortable-list)

## Требования к реализации

Работа ведется в файле `src/index.ts`.

### 1. Интерфейс класса

```typescript
new ProductForm(productId)
```

* **productId**: идентификатор товара (опционально). Если передан, компонент работает в режиме редактирования.

### 2. Публичные свойства

* `element`: корневой DOM-элемент формы.
* `productId`: текущий идентификатор товара.

### 3. Методы

* `render()`: загружает данные и возвращает DOM-элемент формы.
* `save()`: отправляет данные на сервер и диспатчит событие.
* `remove()`: удаляет компонент из DOM.
* `destroy()`: удаляет компонент и очищает обработчики.

### 4. Поведение

* Список изображений реализуется через `SortableList` и поддерживает Drag'n'Drop.
* После сохранения:
  * при редактировании диспатчится событие `product-updated` (detail: id);
  * при создании диспатчится событие `product-saved`.

## Инструкция по запуску

**Запуск в режиме разработки:**
```bash
npm run dev:09:1-product-form-v2
```

**Запуск тестов:**
```bash
npm test 09-tests-for-frontend-apps/1-product-form-v2
```
